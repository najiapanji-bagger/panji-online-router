Static publishing router

One clean domain for many small pages.

Panji Online Router is a lightweight static route index for demos, reports, previews, learning pages, and AI-generated artifacts under /[slug].

Active links

  • /demo/ — Demo page

How it works

  • Create public/[slug]/index.html
  • Add the route to this index
  • Deploy the public folder

Maintainer friendly

  • No database
  • No backend server
  • Git history as audit trail
Public static content only. Do not commit secrets or private data.
