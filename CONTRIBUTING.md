# Contributing

Thank you for considering a contribution to Panji Online Router.

This project is a small static publishing router. Contributions should keep the project simple, transparent, and easy to host.

## Good contributions

- Improve documentation.
- Add route examples.
- Improve accessibility and responsive layout.
- Add small validation scripts.
- Improve maintainer workflow.
- Fix broken links or confusing instructions.

## Before opening a pull request

1. Make sure the route works locally.
2. Do not include secrets or private files.
3. Keep generated pages reasonably small.
4. Use clear route names.
5. Explain what changed and why.

## Local preview

```bash
python3 -m http.server 8080 --directory public
```

Then open:

```text
http://localhost:8080
```

## Security and privacy

Everything in this repository should be treated as public. Do not commit credentials, private screenshots, personal documents, cookies, tokens, or `.env` files.
